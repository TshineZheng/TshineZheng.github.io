Seafile开机启动失败的解决办法

发表于 2016-10-25 分类于 Linux 评论数： 阅读次数：

由于 seafile 依赖于 mysql (或其他数据库程序)，在通过 update-rc.d 设置为开机启动后，开机时可能 mysql 还未启动结束，系统就调用了 seafile 的启动服务，这时就会出现启动失败的情况
