安装MAC10.9.4记录

发表于 2014-07-20 分类于 折腾 评论数： 阅读次数：

我的配置是：
CPU 英特尔 Xeon E3-1230 V2
主板 华擎 B75M-ITX (Intel B75 (Panther Point-M Enahnced))
内存 8 GBytes
显卡 NVIDIA（英伟达） GeForce GTX 660 Ti
硬盘 希捷 ST1000DM003-1CH162
显示器 冠捷 [Unknown Model: AOC2357]
网卡 瑞昱 Semiconductor RTL8168/8111 PCI-E Gigabit Ethernet NIC
声卡 英特尔 Panther Point PCH - High Definition Audio Controller

安装准备：

  1. 下载mac10.9.4懒人版本镜像 http://bbs.pcbeta.com/viewthread-1524752-1-1.html
  2. 下载 DriveGenius
  3. 下载 EasyBCD
  4. 下载 HFS for Windows9.0
  5. 下载 变色龙 http://bbs.pcbeta.com/viewthread-1350886-1-1.html 这里我下载的不是最新版本，最新版 http://bbs.pcbeta.com/viewthread-1518850-1-1.html
  6. 下载 Leaopard hd install helper (安装助手，用来将镜像写入本地硬盘)

开始安装

主要参考 http://bbs.pcbeta.com/viewthread-1169005-1-1.html 这个帖子

  1. 分区 使用Win自带的磁盘管理（计算机》管理》磁盘管理））把硬盘中最后一个盘分出7G作为安装盘。在选择格式化时选择不格式化。
    再分一个MAC系统盘，MAC系统盘最好要保证是逻辑分区的。
  2. 把系统写入硬盘 用管理员身份运行安装助手。载入镜像后，只勾写入。
    这里有个问题，偶尔可能会出现写不进的情况，尝试更换目标分区，然后更改回要写的分区，也许就可以了。
  3. 修改安装盘 由于不明的原因，我的配置需要修改一个驱动文件才可以进入安装界面 http://pan.baidu.com/s/1pJLRPXL 将下载下来的NVDAResman.kext替换镜像盘里的 /System/Library/Extensions/ 同名文件

安装HFS for Windows9.0

有个地方一定要注意：product key的格式必须winodws产品id格式一样，即（XXXXX-XXXXX-XXXXX-XXXXX）,内容随便填，否则安装好HFS for Windows 9.0后还是无法读写苹果分区。

安装变色龙

解压下载下来的变色龙到C盘，运行里面的install.bat

安装引导

  • 运行EasyBcd
  • Add new Entry
  • 选择ISO boot
  • 在Mode选择Run from Disk
  • 然后选择变色龙文件夹里的wowpc.iso
  • Add Entry

开始安装系统

重启电脑后选择刚刚添加的启动项，选择系统镜像盘进入，一路安装即可
安装完之后会发现进不了系统，使用WInPE进入用硬盘工具激活Windows系统分区，再进入Windows系统。
在Windows下将安装盘里的Extra文件夹复制到MAC盘根目录下。重启进入MAC系统

安装驱动

网卡：http://pan.baidu.com/s/1gdkozWf 下载下来直接安装即可
显卡：链接: http://pan.baidu.com/s/1qWBaylu 密码: a7ua
安装成功后先不要重启，做以下操作

  • 在左上角会出现Nvida的图标，点击选择使用Web驱动
  • 修改MAC系统盘/Extra/org.chameleon.Boot.plist里的GraphicsEnabler为No
  • 修改MAC系统盘/Extra/SMBios.plist里的SMboardproduct为Mac-FC02E91DDD3FA6A4
    此步操作很神奇，不这样做就不能双屏显示，不明原因，相关资料：

OKAY重启电脑后就基本完成了
